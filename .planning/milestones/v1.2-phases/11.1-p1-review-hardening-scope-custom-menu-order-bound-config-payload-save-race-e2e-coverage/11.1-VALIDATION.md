## Sampling Rate

- **After every task commit:** Run the relevant quick command ‚Äî `vendor/bin/phpunit -c phpunit-unit.xml.dist --filter <TestClass>` for HARD-01 predicate / HARD-02 caps.
- **After every plan wave:** Run the full suite. **E2E (HARD-03) runs only at the wave boundary** ‚Äî Docker/sandbox blocks per-task E2E (established Phase 9 pattern, STATE.md). The planner MUST schedule a full-suite + Plugin Check gate plan at the end of the phase.
- **Before `/gsd:verify-work`:** Full suite green + Plugin Check 0 errors + phpcs + PHPStan clean.
- **Max feedback latency:** unit < ~10s; E2E deferred to wave boundary by design.
