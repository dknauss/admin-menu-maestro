<objective>
UX-08: make the editor entry reachable and compact on mobile.

UX-08b (PHP): change the admin-bar toggle's VISIBLE label to 'Edit Menu' (enter) / 'Exit' (exit),
keeping the leading dashicon; move the full phrasing 'Edit Admin Menu' / 'Exit Editor' into
`meta.title` so screen readers and the tooltip keep the explicit accessible name.

`#wp-toolbar > ul > li { display:none }` and whitelists only built-in nodes back — so

scoped override that re-shows OUR node (matching core's `#wpadminbar li#id` specificity, no
`!important`), and hide the visible label text at that breakpoint for an icon-only treatment. The
accessible name is preserved by the `<a title="...">` that WP renders from `meta.title`.
