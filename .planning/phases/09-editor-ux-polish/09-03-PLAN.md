buildFirstRunCue (assets/maestro.js:1060+) currently:
- reads window.localStorage.getItem('maestroFirstRunDone') === '1' in a try/catch,
  returns early if seen;
- builds .maestro-firstrun banner + a keyboard-dismissible dismiss button;
- dismiss() sets the localStorage flag and cue.remove().

CHANGE the gate to the Plan-01 seam (keep the try/catch semantics intact — firstRunSeen
returns true on throw, which preserves "skip cue when storage blocked"):
```js
if ( window.maestroLogic.firstRunSeen( window.localStorage ) ) { return; }
```

ADD the pulse after the banner is built/appended:
```js
var firstItem = document.querySelector( '#adminmenu > li.menu-top.maestro-item' );
if ( firstItem ) {
    firstItem.classList.add( 'maestro-firstrun-pulse' );
    firstItem.addEventListener( 'animationend', function onEnd() {
        firstItem.classList.remove( 'maestro-firstrun-pulse' );
        firstItem.removeEventListener( 'animationend', onEnd );
    } );
}
```

EXTEND dismiss() to ALSO clear the pulse (CRITICAL — animationend never fires under
reduced-motion, so the class must be removed here too):
```js
if ( firstItem ) { firstItem.classList.remove( 'maestro-firstrun-pulse' ); }
```
